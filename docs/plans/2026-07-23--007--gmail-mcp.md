## Wzorzec (GitHub → Gmail)

```
Settings UI → OAuth (Google) → integration_oauth_tokens (encrypted)
Agent loop → agent/tools/gmail.py → resolve_access_token("gmail")
         → mcp/gmail/tools.py → Gmail REST API
```

Login Google (`GOOGLE_OAUTH_*`, scopes `email profile`) **≠** integracja Gmail — osobny client + redirect + scopes (jak GitHub login vs GitHub integration).

## Zakres v1

### OAuth

| Element | Wartość |
|---------|---------|
| Provider id | `gmail` |
| Scopes | `openid`, `email`, `profile`, `https://www.googleapis.com/auth/gmail.readonly` |
| Offline | `access_type=offline`, `prompt=consent` (refresh token) |
| Env | `GMAIL_INTEGRATION_OAUTH_CLIENT_ID/SECRET/REDIRECT_URI` |
| Redirect | `{FRONTEND}/settings/integrations/callback/gmail` |

### Narzędzia

| Tool | API |
|------|-----|
| `gmail_search_messages` | `users.messages.list` + `q=` (Gmail query) |
| `gmail_list_messages` | list (labelIds, maxResults) |
| `gmail_get_message` | `users.messages.get` — znormalizowany subject/from/to/date/snippet/body text |

Poza v1: send, modify labels, attachments binary, `gmail_get_thread`.
