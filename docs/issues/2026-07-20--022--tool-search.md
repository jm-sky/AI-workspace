## Cel

Gdy zestaw narzędzi rośnie, nie ładować całego katalogu do każdej tury.

**Model listy narzędzi:**
1. **Narzędzia podstawowe** — zawsze dostępne (m.in. meta-narzędzie *tool search*).
2. **Narzędzia dedykowane agenta** — z profilu / wybranego agenta (kontekst routingu).
3. **Gdy narzędzi jest mało** (poniżej progu) — można przekazać **wszystkie** bez search.

Agent może wołać tool search, gdy potrzebuje narzędzia spoza aktualnie załadowanego podzbioru.
